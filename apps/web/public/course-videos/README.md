# Видео для курса

Курс теперь использует внешние видео сервисы (не нужно загружать файлы в проект).

## Как добавить видео:

### Вариант 1: Google Drive (рекомендуемый)
1. Загрузите видео на Google Drive
2. Откройте видео и нажмите "Получить ссылку"
3. Установите доступ "Всем, у кого есть ссылка"
4. Скопируйте ID файла из ссылки:
   `https://drive.google.com/file/d/FILE_ID/view` → скопируйте FILE_ID
5. Замените YOUR_FILE_ID_X в коде на ваш ID

### Вариант 2: YouTube
1. Загрузите видео на YouTube (можно сделать непубличным)
2. Скопируйте ID видео из ссылки:
   `https://www.youtube.com/watch?v=VIDEO_ID` → скопируйте VIDEO_ID
3. Раскомментируйте YouTube ссылки в коде и замените YOUR_VIDEO_ID_X

## Преимущества:
- Не занимает место в проекте
- Быстрая загрузка видео
- Можно легко обновлять видео без передеплоя
- Автоматическое масштабирование качества
