# Видео для курса

Поместите видеофайлы в эту папку:

- `lesson1.mp4` - Урок 1: Что такое утечки данных и почему это опасно
- `lesson2.mp4` - Урок 2: Как мошенники используют ваши данные  
- `lesson3.mp4` - Урок 3: Признаки звонков мошенников
- `lesson4.mp4` - Урок 4: Что делать, если вам звонят мошенники
- `lesson5.mp4` - Урок 5: Как защитить свои данные в интернете
- `poster.jpg` - Постер для видео (опционально)

## Требования к видео:
- Формат: MP4
- Рекомендуемое разрешение: 1280x720 или выше
- Размер файла: до 100MB на видео для быстрой загрузки

## Как добавить видео:
1. Скопируйте видеофайлы в эту папку
2. Убедитесь, что имена файлов точно соответствуют указанным выше
3. Перезапустите приложение если необходимо
