# Настройка Telegram ботов

В проекте используются ДВА разных Telegram бота:

## 1. Бот для верификации номера телефона
**Переменная окружения:** `TELEGRAM_VERIFICATION_BOT_TOKEN`

**Используется в:**
- `/api/telegram-otp/send` - отправка OTP кодов
- `/api/telegram-webhook` - прием сообщений от пользователей

**Назначение:** Отправка кодов подтверждения для верификации номера телефона

## 2. Основной бот (для других функций)
**Переменная окружения:** `TELEGRAM_BOT_TOKEN`

**Используется в:** Других частях приложения (если есть)

## Как настроить

1. Создайте `.env` файл в `apps/web/`
2. Добавьте обе переменные:

```env
# Бот для верификации телефона
TELEGRAM_VERIFICATION_BOT_TOKEN=your_verification_bot_token_here

# Основной бот (если используется)
TELEGRAM_BOT_TOKEN=your_main_bot_token_here
```

3. Перезапустите приложение

## Webhook для бота верификации

Настройте webhook для бота верификации:
```
https://your-domain.com/api/telegram-webhook
```
