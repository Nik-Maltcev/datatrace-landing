Формат запроса
GET
https://api.datanewton.ru/v1/finance
Параметры запроса
key
string
required
API-ключ.

Example:
mi76aFMdgvml
inn
string
ИНН организации. Должен быть указан, если не указан ОГРН. В случае определения нескольких организаций с этим ИНН (например, при наличии филиалов) будет возвращена информация по головной организации.

Example:
9728006808
Match pattern:
^$|^\d{10}$|^\d{12}$
ogrn
string
ОГРН или ОГРНИП. Должен быть указан, если не указан ИНН.

Example:
1207700223257
Match pattern:
^$|^\d{13}$|^\d{15}$
Пример запроса
https://api.datanewton.ru/v1/finance?key=mi76aFMdgvml&ogrn=1207700223257
